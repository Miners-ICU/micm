## 简介

**MICM** [Micro-Intelligent Campus Management], 意为微智能校园管理, 又叫校织网. 

致力于打造统一的, 智能化的校园管理平台.

>  [背景](/docs/background.md) | [官网]() | [社区](/) 


## 技术选型

### 开发环境

- 操作系统：Windows 10 
- 开发工具：Intellij IDEA
- 数据库：MySQL 8.0.+
- Java SDK：Oracle JDK 1.8

### 部署环境

- 操作系统：Linux Ubuntu Server 18.04 X64
- 虚拟化技术：VMware + Docker

### 项目管理工具

- 项目构建：Maven + Nexus
- 代码管理：Git + Github
- 镜像管理：Docker Registry
